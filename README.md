## 🖐️ 프로젝트 소개
>2023 하반기 대학교 모바일프로젝트 수업에서 진행한,
>안드로이드 스튜디오를 사용해서 어플리케이션을 만드는 프로젝트입니다.
>
>어플리케이션 사용자의 운동과 식단을 지속적으로 유지하는 것을 도울 수 있도록,\n
>간단한 상호작용과 직관적인 UI 를 중점으로 만들었습니다.
